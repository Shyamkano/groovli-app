# 🍒 GROOVLI

**Premium High-Performance Music Player for Android**
*Built with React Native & Expo*

![Groovli Banner](assets/image.png)

## 🚀 Experience Music Reimagined
Groovli is more than just a music player—it's a high-performance audio engine designed for modern listeners. Focused on minimalist aesthetics and zero-interruptions, Groovli gives you direct access to millions of tracks worldwide with instant response times.

### **✨ Key Features**
- **⚡ Super-Fast Streaming**: Optimized network handling for <0.4s response times.
- **🔍 Global Discovery**: Instant access to millions of tracks, artists, and albums using the JioSaavn API.
- **🎨 Cherry-Themed UI**: A stunning, minimalist dark interface designed for focus.
- **📜 Smart Lyrics**: Real-time synchronized lyrics for an immersive listening experience.
- **📱 Responsive Layout**: Fully optimized for every Android screen size.
- **🎧 Seamless Queue**: One-tap access to your playback history and upcoming tracks.

## 🛠️ Tech Stack
Groovli leverages the latest in mobile development to deliver a premium experience:

- **Frontend**: [React Native](https://reactnative.dev/) with [Expo](https://expo.dev/)
- **Backend API**: Custom [Node.js](https://nodejs.org/) interface for the [JioSaavn](https://www.jiosaavn.com/) engine
- **State Management**: React Context API (Player, Auth, Toast)
- **Navigation**: React Navigation (Native Stack & Bottom Tabs)
- **Styling**: Native components with a custom "Cherry" design system
- **Deployment**: EAS (Expo Application Services)

## 📦 Getting Started

### **Prerequisites**
- [Node.js](https://nodejs.org/) (LTS)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- [EAS CLI](https://docs.expo.dev/build/setup/) (for production builds)

### **Installation**
1. **Clone the repository**
   ```bash
   git clone https://github.com/Shyamkano/groovli-app.git
   cd groovli-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start Development**
   ```bash
   npx expo start
   ```

## 🏗️ Building for Production
Groovli uses EAS for optimized distribution. To generate a high-quality APK:
```bash
eas build -p android --profile production
```
